# Excel Dashboard Portfolio

This repository contains interactive dashboards created using Microsoft Excel. The dashboards are designed to analyze business data, track KPIs, and provide insights through visual reports.

## Project Overview

The goal of this repository is to showcase data analysis and visualization skills using Excel. Each dashboard focuses on transforming raw data into meaningful insights using charts, pivot tables, and KPI indicators.

## Tools & Features Used

* Microsoft Excel
* Pivot Tables & Pivot Charts
* Data Cleaning & Data Formatting
* KPI Cards
* Slicers and Filters
* Conditional Formatting




## Key Insights

* Identified sales trends and performance metrics
* Analyzed category and region-wise performance
* Tracked revenue, profit, and order distribution

## Skills Demonstrated

* Data Analysis
* Business Reporting
* Dashboard Design
* Data Visualization
